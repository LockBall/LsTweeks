# Sound Levels Assets

Place reduced-volume replacement sounds for the Sound Levels module here.

Current planned filenames:
- `test_sound_shush.ogg`
- `test_sound_shusher.ogg`
- `test_sound_shushest.ogg`
- `dungeon_ready_shush.ogg`
- `dungeon_ready_shusher.ogg`
- `dungeon_ready_shushest.ogg`

The module mutes known original Blizzard FileDataIDs, then plays these module-owned files for the selected preset.
